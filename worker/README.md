# Worker Progress Tracker

A simple application to track and monitor worker productivity, task completion, and project progress across teams. Useful for managers to assign tasks, log status updates, and generate performance reports.

## Features

- 👷 Add and manage worker profiles
- ✅ Assign tasks and track completion status (Pending, In Progress, Completed)
- 📅 Log daily/weekly progress updates
- 📊 Generate progress reports by worker, task, or project
- ⏱️ Track time spent on tasks
- 🚧 Flag delays or blockers
- 🔔 Notifications/reminders for pending tasks
